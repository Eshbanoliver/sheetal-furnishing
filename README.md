# Sheetal Furnishing - Luxury Interior Solutions



**Live Demo:** [https://bisque-alligator-639001.hostingersite.com/](https://bisque-alligator-639001.hostingersite.com/)

## Description

Modern, responsive website for **Sheetal Furnishing** — a premium luxury furnishing showroom based in Jaipur, Rajasthan (established in 2007).

The site showcases high-end interior products and services including:

- Premium carpets & rugs
- Bespoke curtains & motorized blinds
- Executive & luxury sofas
- Fine bedding collections
- Designer upholstery fabrics

It highlights the brand's 17+ years of legacy, commitment to quality, custom tailoring, professional installation, and partnerships with top brands like D'Decor and many others.

## Features

- Clean, elegant and mobile-responsive design
- Bootstrap 5 framework
- Smooth scrolling & modern layout
- Sections: Hero banner, About, Services/Products, Why Choose Us, Testimonials, FAQ, Contact & Location info
- Call-to-action buttons for consultations and directions
- Professional color scheme (luxury feel: gold/cream/dark accents)

## Tech Stack

- **HTML5**
- **CSS3** (custom + utility classes)
- **JavaScript** (vanilla + minor interactivity)
- **Bootstrap 5** (responsive grid, components, utilities)

No heavy frameworks, no build tools — pure front-end static website.
